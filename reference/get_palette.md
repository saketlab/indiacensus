# Get color palette

Get color palettes for maps. Available palettes: "viridis", "red_blue",
"blue_red", "greens", "oranges", "reds", "blues", "purple_green".

## Usage

``` r
get_palette(name, reverse = FALSE)
```

## Arguments

- name:

  Name of the palette

- reverse:

  Logical. Reverse direction?

## Value

A character vector of colors
