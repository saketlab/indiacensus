# Census 1961 literacy data

District-level literacy rates from the 1961 Census of India.

## Usage

``` r
census_1961_literacy
```

## Format

A tibble with 347 rows and 8 columns:

- year:

  Census year (1961)

- state:

  Name of the state

- state_name_harmonized:

  Harmonized state name for joining across datasets

- district:

  Name of the district

- state_district_code:

  Combined state-district code

- literacy_total:

  Total literacy rate (percent)

- literacy_male:

  Male literacy rate (percent)

- literacy_female:

  Female literacy rate (percent)

## Source
