# Indian States Lookup

A lookup table of Indian states with their codes, abbreviations, and
regions.

## Usage

``` r
india_states
```

## Format

A tibble with 36 rows and 5 columns:

- state_code:

  Numeric state code (Census 2011 codes)

- state_name:

  Official state name

- state_name_harmonized:

  Harmonized state name for joining across datasets

- state_abbr:

  Two-letter state abbreviation

- region:

  Geographic region (North, South, East, West, Central, Northeast,
  Islands)
